# 博弈论

## sg函数

对于状态x和它的后继状态$$y_1,y_2,...,y_k$$

$$SG(x)=mex\{SG(y_1),SG(y_2),...,SG(y_k)\}$$

$$SG(x)\neq 0$$必胜，$$=0$$必败

起点有n个有向图$$s_1,s_2,...,s_n$$，当且仅当

$$SG(s_1)\oplus SG(s_2)...\oplus SG(s_n)\neq 0$$时，先手必胜
